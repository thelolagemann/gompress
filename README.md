# gompress

gompress is a small CLI utility for working with compressed data formats. Currently supported formats are 

| Format | Compress | Decompress |
|--------|----------|------------|
| brotli | ✓        | ✓          |
| bzip2  | ✓        | ✓          |
| gzip   | ✓        | ✓          |
| lz4    | ✓        | ✓          |
| lzma2  | ✓        | ✓          |
| snappy | ✓        | ✓          |
| zlib   | ✓        | ✓          |
| zstd   | ✓        | ✓          |
